![Logger](https://github.com/orhanobut/logger/raw/master/images/logger-logo.png)


有些手机不显示debug信息可以这样

需要打开华为手机的相关Log开关，具体步骤如下：

1. 进入拨号界面输入：
    ```*#*#2846579#*#*```
2. 依次选择ProjectMenu---后台设置----LOG设置---LOG开关 点击打开, 然后在LOG级别选VERBOSE
3. 重新启动手机
